# 📊 Обзор проекта

Полный обзор Django MVT сайта-портфолио для врача-уролога Максудова Абдурахмана.

## 🎯 Цель проекта

Создание профессионального сайта-портфолио врача-уролога с возможностью:
- Демонстрации медицинских достижений и публикаций
- Предоставления информации об услугах
- Приёма записей на прием
- Публикации статей в медицинском блоге
- Заказа медицинских книг

### 📋 Модели данных

| Модель | Описание | Поля |
|--------|----------|------|
| Profile | Профиль врача | ФИО, фото, биография, контакты, клиника, соцсети |
| Service | Услуги | Название, описание, цена, длительность |
| Publication | Публикации | Заголовок, авторы, год, журнал, DOI, PDF |
| Project | Клинические случаи | Название, описание, даты, роль, результаты |
| BlogPost | Статьи блога | Заголовок, **WYSIWYG содержание**, изображение, категория |
| ServiceOrder | Записи на прием | Пациент, услуга, возраст, сообщение, статус |
| Achievement | Достижения | Название, дата, организация, сертификат |
| Testimonial | Отзывы | Пациент, текст, рейтинг |
| ContactMessage | Сообщения | Имя, email, тема, текст |
| **Book** 📚 | **Книги** | Название, автор, **WYSIWYG описание**, обложка, **PDF**, цена |
| **BookOrder** 📦 | **Заказы книг** | Книга, клиент, адрес, количество, стоимость |

### 🎨 Дизайн

- **Framework**: Bootstrap 5
- **Иконки**: Font Awesome 6
- **Шрифты**: Roboto + Playfair Display (Google Fonts)
- **Цветовая схема**: Медицинская (синий/белый)
- **Адаптивность**: Mobile-first подход
- **Анимации**: Hover эффекты, плавные переходы

### 🛠 Технологии


## 💻 Страницы сайта

### 1. Главная страница (/)
- Hero секция с фото и кратким описанием
- Обзор услуг (6 карточек)
- Избранные публикации (3 шт)
- Отзывы пациентов
- Последние статьи блога
- Call-to-action секции

### 2. О враче (/about/)
- Полная биография
- Фото профиля
- Образование и квалификация
- Контактная информация
- Социальные сети
- Достижения и награды (timeline)

### 3. Услуги (/services/)
- Карточки медицинских услуг с описанием
- Цены и длительность
- Иконки для визуализации
- Кнопки записи

### 4. Клинические случаи (/portfolio/)
- Список клинических случаев
- Фильтр по статусу (в процессе/завершен)
- Детали каждого случая
- Роль, организация, результаты

### 5. Публикации (/publications/)
- Список всех публикаций
- Фильтры: тип, год
- Поиск по названию/авторам/ключевым словам
- Пагинация
- Ссылки на DOI, PDF

### 6. Книги (/books/) 📚
- Каталог книг с обложками
- Фильтрация по году издания
- Детальная страница с **PDF-просмотром**
- Издательская информация (ISBN, издательство, страницы)
- Цены
- Кнопка "Заказать книгу"

### 7. Блог (/blog/)
- Карточки статей с изображениями
- **WYSIWYG форматирование** контента
- Фильтр по категориям
- Теги
- Счетчик просмотров
- Пагинация

### 8. Контакты (/contact/)
- Контактная информация
- Форма обратной связи
- Социальные сети
- Адрес клиники

### 9. Запись на прием (/order/)
- Форма записи на прием
- Выбор услуги
- Валидация полей
- Email уведомления

### 10. Заказ книги (/order-book/)
- Форма заказа с выбором книги
- Количество экземпляров
- Адрес доставки
- Автоматический расчет стоимости
- Email уведомления
